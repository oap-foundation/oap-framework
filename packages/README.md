# OAP Framework

The Open Agent Protocol (OAP) Framework is a collection of PHP libraries for building autonomous agents that can discover, negotiate, and transact with each other.

## Packages

- **[oaep](packages/oaep)**: Open Agent Exchange Protocol (Identity, Security, Session).
- **[oacp](packages/oacp)**: Open Agent Commerce Protocol (Negotiation, Orders).
- **[discovery](packages/discovery)**: Agent Discovery Protocol (Capabilities, Trust).
- **[transport](packages/transport)**: Physical Transport Layer (HTTP, etc.).
- **[oapp](packages/oapp)**: Open Agent Payment Protocol (Payment Abstraction).

## Skeleton

- **[skeleton](skeleton)**: A reference implementation of an OAP Agent Kernel (Buyer/Seller).

## Installation

To use the framework, we recommend starting with the skeleton or requiring individual packages via Composer.
